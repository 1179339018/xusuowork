# 静态资源目录

此目录用于存放静态资源文件，包括：

## tabbar 图标
- `tabbar/home.png` - 首页图标（未选中）
- `tabbar/home-active.png` - 首页图标（选中）
- `tabbar/task.png` - 任务图标（未选中）
- `tabbar/task-active.png` - 任务图标（选中）
- `tabbar/add.png` - 录入图标（未选中）
- `tabbar/add-active.png` - 录入图标（选中）
- `tabbar/street.png` - 街道图标（未选中）
- `tabbar/street-active.png` - 街道图标（选中）
- `tabbar/mine.png` - 我的图标（未选中）
- `tabbar/mine-active.png` - 我的图标（选中）

## 图标规格
- 尺寸：81px × 81px
- 格式：PNG（支持透明背景）
- 建议使用蓝色主题色 #1677ff

## 其他资源
- 可以在此目录添加其他图片、字体等静态资源
